# 网站信息架构设计

## 一、网站定位

- **类型**: 科技公司展示型官网
- **目标**: 品牌形象展示 + 业务介绍 + 客户转化
- **受众**: B端客户、投资者、合作伙伴、求职者
- **技术栈**: Vue3 + CMS后台 + 多语言支持

## 五、URL结构设计

### 5.1 URL命名规范

```
# 多语言前缀
/zh/    # 简体中文
/en/    # 英文
/jp/    # 日文
/ko/    # 韩文

# 页面URL示例
/zh/                    # 首页
/zh/about/company       # 公司简介
/zh/products            # 产品中心
/zh/products/ai-platform # 产品详情
/zh/solutions/industry/finance # 行业方案
/zh/news/2026/03/11/article-slug # 新闻详情
```

### 5.2 URL设计原则

1. **语义化**: 使用英文单词，避免数字ID
2. **层级清晰**: 通过路径体现内容层级
3. **简洁性**: 避免过长URL
4. **多语言**: 语言前缀 + 内容路径
5. **SEO友好**: 包含关键词，使用连字符分隔

## 六、CMS内容模型设计

### 6.1 内容类型

| 内容类型 | 字段 | 用途 |
|----------|------|------|
| **页面** | 标题、内容、SEO信息、模板 | 静态页面管理 |
| **产品** | 名称、描述、图片、规格、分类 | 产品展示 |
| **解决方案** | 标题、行业、场景、内容、案例 | 方案展示 |
| **新闻** | 标题、摘要、内容、分类、标签、发布时间 | 新闻发布 |
| **案例** | 客户名称、行业、挑战、方案、效果、图片 | 案例展示 |
| **团队成员** | 姓名、职位、照片、简介 | 团队介绍 |
| **招聘信息** | 职位名称、部门、地点、要求、描述 | 招聘管理 |

### 6.2 多语言支持

```
内容表结构:
- id: 内容ID
- type: 内容类型
- slug: URL标识
- translations: {
    zh: { title, content, ... },
    en: { title, content, ... },
    ...
  }
- created_at
- updated_at
```

## 七、用户体验流程

### 7.1 访客旅程

```
认知阶段 → 兴趣阶段 → 考虑阶段 → 决策阶段
    │           │           │           │
    ▼           ▼           ▼           ▼
  首页      产品/方案    案例/详情    联系/咨询
  广告点击    深入了解    对比评估    留下线索
```

### 7.2 关键转化路径

1. **首页 → 产品页 → 咨询表单**
2. **首页 → 解决方案 → 案例 → 咨询**
3. **文章/SEO → 产品页 → 咨询**
4. **关于我们 → 团队/文化 → 招聘申请**

## 八、技术实现建议

### 8.1 路由配置 (Vue Router)

```javascript
const routes = [
  {
    path: '/:lang(zh|en|jp|ko)?',
    component: Layout,
    children: [
      { path: '', component: Home },
      { path: 'about/:page?', component: About },
      { path: 'products/:slug?', component: Products },
      { path: 'solutions/:category?/:slug?', component: Solutions },
      { path: 'news/:category?/:slug?', component: News },
      { path: 'contact/:page?', component: Contact },
    ]
  }
]
```

## 九、SEO与性能考虑

### 9.1 SEO策略

- 每页面独立的Title、Description
- 语义化HTML标签
- 结构化数据标记
- XML站点地图
- 友好的URL结构
- 多语言hreflang标签

### 9.2 性能优化

- 路由懒加载
- 图片懒加载 + WebP格式
- 静态页面生成(SSG)
- CDN加速
- 资源压缩
