# 重庆启云网络科技有限公司官网 - 技术架构文档

## 📋 项目概述

## 2. 技术选型说明

### 2.1 前端技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue | 3.4.x | 渐进式 JavaScript 框架，Composition API |
| TypeScript | 5.3.x | 类型安全的 JavaScript 超集 |
| Vite | 5.0.x | 下一代前端构建工具，极速冷启动 |
| Vue Router | 4.2.x | 官方路由管理器 |
| Pinia | 2.1.x | 状态管理库，Vuex 替代品 |
| Element Plus | 2.5.x | UI 组件库 |
| Tailwind CSS | 3.4.x | 原子化 CSS 框架 |
| Axios | 1.6.x | HTTP 客户端 |
| VueUse | 10.7.x | Vue 组合式工具库 |

### 2.2 后端技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Java | 17 | LTS 版本，性能优化 |
| Spring Boot | 3.2.x | 快速开发框架 |
| Spring Security | 6.2.x | 安全认证框架 |
| MyBatis Plus | 3.5.x | ORM 框架增强版 |
| MySQL | 8.0 | 关系型数据库 |
| Redis | 7.x | 缓存数据库 |
| JWT | 0.12.x | Token 认证 |
| Lombok | 1.18.x | 代码简化工具 |
| Swagger | 2.3.x | API 文档 |

### 2.3 选型理由

**Vue3 + Vite:**
- Vite 提供极速的开发体验，冷启动 < 300ms
- Vue3 Composition API 更好的逻辑复用和类型推断
- 更好的 TypeScript 支持

**Pinia vs Vuex:**
- 更简洁的 API 设计
- 完整的 TypeScript 支持
- 模块化的 store 设计

**Element Plus:**
- 企业级 UI 组件库
- 完善的文档和社区支持
- 支持按需引入

**Tailwind CSS:**
- 原子化 CSS，减少样式文件体积
- 响应式设计友好
- 开发效率高

### 4.3 代码示例

```java
@RestController
@RequestMapping("/api/v1/cases")
@Tag(name = "案例管理", description = "案例展示相关接口")
public class CaseController {

    @Autowired
    private CaseService caseService;

    @GetMapping
    @Operation(summary = "获取案例列表")
    public Result<PageResult<CaseVO>> list(
            @RequestParam(defaultValue = "1") Integer page,
            @RequestParam(defaultValue = "10") Integer size,
            @RequestParam(required = false) String category) {
        return Result.success(caseService.list(page, size, category));
    }

    @GetMapping("/{id}")
    @Operation(summary = "获取案例详情")
    public Result<CaseDetailVO> detail(@PathVariable Long id) {
        return Result.success(caseService.getDetail(id));
    }
}
```

```sql
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES career_category(id)
) COMMENT '职位表';

-- 职位申请表
CREATE TABLE job_application (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    career_id BIGINT NOT NULL COMMENT '职位ID',
    name VARCHAR(50) NOT NULL COMMENT '姓名',
    phone VARCHAR(20) NOT NULL COMMENT '电话',
    email VARCHAR(100) COMMENT '邮箱',
    resume_url VARCHAR(500) COMMENT '简历URL',
    message TEXT COMMENT '留言',
    status TINYINT DEFAULT 0 COMMENT '状态：0-待处理 1-已查看 2-已联系 3-已拒绝',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (career_id) REFERENCES career(id)
) COMMENT '职位申请表';

-- 留言表
CREATE TABLE contact_message (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL COMMENT '姓名',
    phone VARCHAR(20) COMMENT '电话',
    email VARCHAR(100) COMMENT '邮箱',
    company VARCHAR(100) COMMENT '公司名称',
    subject VARCHAR(200) COMMENT '主题',
    content TEXT NOT NULL COMMENT '内容',
    status TINYINT DEFAULT 0 COMMENT '状态：0-未读 1-已读 2-已回复',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) COMMENT '留言表';
```

### 6.2 部署方式

**方案一：Docker Compose（推荐开发/测试）**

```yaml
# docker-compose.yml
version: '3.8'
services:
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx/nginx.conf:/etc/nginx/nginx.conf
      - ./frontend/dist:/usr/share/nginx/html
    depends_on:
      - backend

  backend:
    build: ./backend
    ports:
      - "8080:8080"
    environment:
      - SPRING_PROFILES_ACTIVE=prod
      - MYSQL_HOST=mysql
      - REDIS_HOST=redis
    depends_on:
      - mysql
      - redis

  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: your_password
      MYSQL_DATABASE: qiyun_website
    volumes:
      - mysql_data:/var/lib/mysql
      - ./database/schema.sql:/docker-entrypoint-initdb.d/01-schema.sql
      - ./database/data.sql:/docker-entrypoint-initdb.d/02-data.sql

  redis:
    image: redis:7-alpine
    volumes:
      - redis_data:/data

volumes:
  mysql_data:
  redis_data:
```

**方案二：云服务器部署（推荐生产环境）**

- **服务器**: 阿里云 ECS / 腾讯云 CVM (2核4G起步)
- **数据库**: 阿里云 RDS MySQL / 腾讯云 Cloud Database
- **文件存储**: 阿里云 OSS / 腾讯云 COS
- **CDN**: 阿里云 CDN / 腾讯云 CDN
- **域名**: 配置 SSL 证书 (Let's Encrypt)
