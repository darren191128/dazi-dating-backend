# 律所网站CMS后台管理系统功能规范

> 创建时间：2026-03-12
> 适用项目：若山律师事务所官网
> 技术栈：Vue3 + Java

## 核心功能模块

### 1. 系统管理
| 功能 | 说明 |
|------|------|
| 用户管理 | 管理员账号增删改查、角色分配 |
| 角色权限 | 超级管理员、内容编辑、普通管理员 |
| 登录认证 | JWT Token、登录日志、密码修改 |
| 系统设置 | 网站名称、Logo、SEO设置、备案号 |

### 2. 内容管理（CMS核心）
| 功能 | 说明 |
|------|------|
| 轮播图管理 | 首页Banner增删改查、排序、上下线 |
| 文章管理 | 新闻资讯、法律文章的发布/编辑/删除 |
| 分类管理 | 文章分类、服务分类的增删改查 |
| 页面管理 | 关于我们、服务领域等静态页面编辑 |
| 友情链接 | 合作律所、相关链接管理 |

### 3. 业务管理
| 功能 | 说明 |
|------|------|
| 律师管理 | 律师信息、头像、简介、专业领域 |
| 案例管理 | 成功案例的增删改查、分类、推荐 |
| 服务管理 | 法律服务项目、价格、介绍 |
| 招聘管理 | 招聘信息发布、简历接收 |

### 4. 客户互动
| 功能 | 说明 |
|------|------|
| 咨询管理 | 在线咨询表单、留言查看回复 |
| 预约管理 | 预约咨询的查看、处理、状态跟踪 |
| 评价管理 | 客户评价审核、展示控制 |

### 5. 数据统计
| 功能 | 说明 |
|------|------|
| 访问统计 | PV/UV、页面访问量、来源分析 |
| 咨询统计 | 咨询量、转化率、热门服务 |
| 内容统计 | 文章阅读量、热门文章排行 |

### 6. 系统工具
| 功能 | 说明 |
|------|------|
| 文件管理 | 图片上传、文件管理、CDN同步 |
| 缓存管理 | 清除缓存、缓存策略设置 |
| 日志管理 | 操作日志、错误日志查看 |
| 数据备份 | 数据库备份、恢复 |

## API权限控制

### 权限分级
| 角色 | 权限范围 |
|------|---------|
| 超级管理员 | 所有权限 |
| 内容编辑 | 内容管理 + 业务管理 |
| 普通管理员 | 查看权限 + 部分内容管理 |

### API设计原则
- 所有API需要JWT认证
- 敏感操作需要二次确认
- 操作日志记录
- 接口限流防刷
