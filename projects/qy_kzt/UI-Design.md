## 1. 设计概述

### 1.1 设计理念
- **简洁高效**: 以用户任务为核心，减少认知负担
- **一致性**: 统一的设计语言，跨端体验一致
- **可扩展**: 组件化设计，支持业务快速迭代
- **本土化**: 符合国内用户使用习惯
- **业务区分**: 四大业务线视觉差异化，便于用户识别

### 1.2 设计目标
- 打造专业、可信赖的区域服务平台形象
- 支持多端统一体验（PC、APP、小程序）
- 后台系统高效易用，降低运营学习成本
- 符合美团/饿了么/淘宝/京东等行业标杆标准

### 1.3 用户角色对应界面

| 用户角色 | 终端类型 | 主要界面 |
|---------|---------|---------|
| 消费者 | APP/小程序/H5 | C端消费者端 |
| 商家 | PC Web | 商家后台工作台 |
| 供应商 | PC Web | 供应商后台工作台 |
| 采购方 | PC Web | 采购方后台工作台 |
| 金厨学校 | PC Web | 学校管理后台 |
| 平台运营 | PC Web | 运营管理后台 |
| 骑手 | APP | 骑手端APP |

## 3. 组件规范

### 3.1 按钮

#### 主按钮
- 背景: `#1677FF`，文字白色，高度44px，圆角4px
- 悬停: 背景 `#4096FF`，阴影加深
- 禁用: 背景 `#BFBFBF`

#### 次按钮
- 背景白色，边框1px solid `#D9D9D9`
- 悬停: 边框和文字变为主色

#### 文字按钮
- 背景透明，文字主色
- 悬停: 背景 `#E6F4FF`

#### 危险按钮
- 背景 `#F5222D`，文字白色

### 3.2 输入框
- 高度44px，边框1px solid `#D9D9D9`，圆角4px
- 聚焦: 边框 `#1677FF`，阴影 `0 0 0 2px rgba(22,119,255,0.2)`
- 错误: 边框 `#F5222D`

### 3.3 卡片
- 背景白色，圆角8px，阴影1
- 内边距16-24px
- 悬停: 阴影2，上移2px

### 3.4 表格
- 表头背景 `#FAFAFA`，文字 `#262626`
- 行高56px，单元格内边距16px
- 斑马纹: 偶数行 `#FAFAFA`
- 悬停: 背景 `#E6F4FF`

### 3.5 导航

#### 顶部导航（C端）
- 高度56px（移动端）/ 64px（PC端）
- 背景白色，底部阴影

#### 底部Tab导航（移动端）
- 高度56px + 安全区
- 5个入口：首页/分类/购物车/订单/我的

#### 侧边导航（后台）
- 宽度200px，背景 `#001529`
- 选中: 背景 `#1677FF`

### 3.6 标签

#### 状态标签
| 状态 | 背景 | 文字 |
|------|------|------|
| 成功 | `#F6FFED` | `#52C41A` |
| 警告 | `#FFFBE6` | `#FAAD14` |
| 错误 | `#FFF2F0` | `#F5222D` |
| 进行中 | `#E6F4FF` | `#1677FF` |

### 3.7 弹窗与反馈
- 对话框圆角12px，阴影3
- 消息提示顶部居中，3秒自动关闭
- 二次确认宽度240px

### 3.8 加载与空状态
- 骨架屏背景 `#F5F5F5`，渐变动画
- 空状态插图120px，标题16px，描述14px
- 错误状态插图200px，带错误码

## 7. 页面原型描述

### 7.1 关键页面布局描述

#### C端首页布局
- **顶部栏**: 固定定位，高度56px，包含定位、搜索框、消息入口
- **Banner区**: 轮播图，高度180px，圆角12px，自动轮播5秒
- **分类入口**: 4个主要业务入口，图标48px，圆角16px，2行2列布局
- **推荐区**: 多模块组合，每个模块包含标题+横向滚动列表
- **底部Tab**: 固定定位，高度56px+安全区，5个入口

#### 商家详情页布局
- **头部区**: 商家大图200px，圆角0（通顶），渐变遮罩
- **信息区**: 商家名称、评分、销量、配送信息，白色背景卡片
- **优惠券区**: 横向滚动，优惠券卡片宽度280px
- **内容区**: 左右分栏，左侧分类栏80px固定，右侧菜品列表
- **购物车栏**: 固定底部，高度60px，显示金额和结算按钮

#### 后台列表页布局
- **筛选区**: 卡片容器，内边距24px，多列筛选条件
- **操作区**: 左侧批量操作，右侧导出/新增按钮
- **表格区**: 全宽表格，表头固定，斑马纹，悬停效果
- **分页区**: 底部居中，显示总条数和分页控件

### 7.2 核心交互流程说明

#### 下单流程（外卖）
1. **浏览商家**: 首页/分类 → 商家列表 → 商家详情
2. **选购商品**: 选择分类 → 浏览菜品 → 点击加减选择数量
3. **查看购物车**: 点击购物车栏 → 查看已选商品 → 可调整数量
4. **确认订单**: 点击去结算 → 确认地址 → 选择配送时间 → 选择优惠券
5. **支付**: 选择支付方式 → 调起支付 → 支付成功 → 订单详情

#### 商家入驻流程
1. **提交申请**: 填写基本信息 → 上传资质 → 提交审核
2. **审核中**: 显示审核进度 → 可补充资料
3. **审核通过**: 设置店铺信息 → 上传菜品 → 开启营业
4. **审核拒绝**: 查看原因 → 修改资料 → 重新提交

#### 售后申请流程
1. **申请入口**: 订单详情 → 申请售后
2. **选择类型**: 仅退款/退货退款
3. **填写原因**: 选择原因 → 填写说明 → 上传凭证
4. **提交审核**: 等待商家处理
5. **处理结果**: 商家同意/拒绝 → 执行退款/重新申请

### 7.3 状态变化设计

#### 订单状态流转
```
待支付 → 已支付 → 待接单 → 待发货 → 配送中 → 待收货 → 已完成
  ↓        ↓         ↓         ↓         ↓         ↓
已取消   退款中    已拒绝    已取消    异常      售后中    已评价
```

**状态标签颜色:**
- 待支付: 警告色 `#FAAD14`
- 已支付/待接单/待发货: 信息色 `#1677FF`
- 配送中: 主色 `#1677FF` + 动态效果
- 待收货: 警告色 `#FAAD14`
- 已完成: 成功色 `#52C41A`
- 已取消/已拒绝: 错误色 `#F5222D`

#### 审核状态显示
- 待审核: 警告标签 + 时钟图标
- 审核中: 信息标签 + 加载图标
- 已通过: 成功标签 + 对勾图标
- 已拒绝: 错误标签 + 叉号图标
