## 六、技术栈

### 后端
- **框架**: Spring Boot 3.2.0
- **微服务**: Spring Cloud Gateway + Nacos
- **数据访问**: Spring Data JPA + MyBatis Plus
- **数据库**: MySQL 8.0
- **缓存**: Redis
- **安全**: JWT + 自定义过滤器
- **音视频**: 腾讯云TRTC
- **其他**: Lombok、Hibernate Validator

### 前端
- **框架**: uni-app
- **语法**: Vue3 Composition API
- **组件库**: uview-plus
- **状态管理**: Pinia
- **构建工具**: Vite
