## 四、VIP特权详情

| 特权编码 | 特权名称 | 说明 |
|---------|---------|------|
| unlimited_swipe | 无限滑动 | 每日滑动次数无限制 |
| visitor_invisible | 访客隐身 | 访问他人主页不留记录 |
| advanced_filter | 高级筛选 | 解锁身高/学历/收入筛选 |
| read_receipt | 已读回执 | 查看消息是否已读 |
| vip_badge | 专属标识 | VIP专属头像框/标识 |
| priority_recommend | 优先推荐 | 在推荐列表中优先展示 |
| view_visitor | 查看访客 | 查看谁访问过我的主页 |
| super_like | 超级喜欢 | 每日3次超级喜欢 |
