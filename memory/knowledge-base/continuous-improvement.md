## 改进机制

### 1. 问题记录（实时）
- 每个项目执行中遇到的问题立即记录
- 分类：技术问题、流程问题、沟通问题、工具问题
- 记录到：`memory/knowledge-base/issues/YYYY-MM-DD-问题简述.md`

### 2. 定期总结（每项目结束）
- 项目复盘会议（虚拟）
- 总结成功经验和失败教训
- 更新到：`memory/knowledge-base/project-lessons-learned.md`

### 3. 知识共享（实时同步）
- 新问题立即通知相关智能体
- 更新共享记忆 `memory/db/memory.json`
- 确保所有智能体都能访问最新经验

### 4. 流程优化（持续）
- 根据问题反馈优化工作流程
- 更新 `memory/knowledge-base/project-workflow.md`
- 定期回顾并改进
