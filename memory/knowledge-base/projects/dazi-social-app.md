# 搭子交友微信小程序

## 项目概述
- **项目名称:** 搭子交友
- **创建时间:** 2026-03-22
- **类型:** 微信小程序
- **目标:** 基于地理位置和兴趣匹配的社交活动平台

---

## 核心功能

### 1. 用户匹配系统
- **附近的人** - 基于地理位置发现附近用户
- **兴趣匹配** - 根据共同兴趣推荐
- **星座匹配** - 星座相容性算法
- **生肖匹配** - 生肖配对推荐
- **综合推荐** - 多维度智能推荐算法

### 2. 活动系统

#### 活动类型（线下）
| 类型 | 说明 |
|------|------|
| 吃喝玩乐 | 聚餐活动 |
| 户外游玩 | 郊游、旅游 |
| 亲子活动 | 陪孩子玩耍 |
| 户外运动 | 羽毛球、篮球等 |
| 相亲交友 | 组织相亲聚会 |

#### 付费模式
| 模式 | 说明 | 示例 |
|------|------|------|
| AA制 | 人均分摊 | 10人聚餐1000元 = 人均100元 |
| 男A女免 | 男性付费，女性免费 | - |
| 请客 | 发起人全额支付 | - |
| 免费 | 无需付费 | - |

#### 活动形式
- 线上活动
- 线下活动

## 技术栈

### 前端
- **框架:** UniApp + Vue3
- **UI库:** uview-plus

### 后端
- **语言:** Java
- **框架:** Spring Cloud Alibaba

### 数据层
- **数据库:** MySQL 8.0
- **缓存:** Redis
- **消息队列:** RabbitMQ
