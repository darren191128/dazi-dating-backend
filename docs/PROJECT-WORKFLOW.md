# 项目开发标准流程

**制定日期**: 2026-03-12  
**制定人**: 老板  
**执行人**: 云壹及团队  

---

## 🔄 标准开发流程

### 阶段一：项目启动 (云壹 → 云捌)

**触发条件**: 云壹接收到老板任务

**执行步骤**:
1. 云壹通知云捌进行项目分析
2. 云捌进行完整的项目分析，必须包含：
   - ✅ 前台界面及操作动效
   - ✅ 后台管理系统界面及功能
   - ✅ 数据库设计
   - ✅ API接口设计
3. 云捌输出完整的PRD文档

**交付物**:
- 产品需求文档 (PRD)
- 功能模块规划
- 技术架构设计
- 数据库设计文档
- API接口文档

---

### 阶段二：UI设计 (云捌 → 云肆)

**触发条件**: 云捌完成项目分析

**执行步骤**:
1. 云捌将整体需求交给云肆
2. 云肆进行UI界面设计

**交付物** (必须包含):
- ✅ UI设计标准 (色彩、字体、间距、组件规范)
- ✅ UI设计图 (所有页面)
- ✅ 动效设计说明
- ✅ 响应式适配方案

---

### 阶段三：开发前确认 (云肆 → 云伍/云陆)

**触发条件**: 云肆完成UI设计

**执行步骤**:
1. 云伍和云陆先了解云肆和云捌的成果
2. 检查是否符合需求
3. **如果符合**: 开始制作
4. **如果不符合**: 
   - 通知云壹
   - 云壹统筹与云捌、云肆校对
   - 确认无误后重新通知云伍和云陆

---

### 阶段四：开发阶段 (云伍 + 云陆 并行)

#### 云伍 - 前端开发

**必须完成**:
- ✅ 前台PC端界面
- ✅ 手机自适应 (移动端)
- ✅ 后台管理界面 (CMS)
- ✅ 所有动效实现
- ✅ 自检查：是否有缺失

#### 云陆 - 后端开发

**必须完成**:
- ✅ 后台功能API
- ✅ 前台联动API
- ✅ 数据库实现
- ✅ 管理员登录及权限
- ✅ 自检查：API是否正确、后台功能是否可操作

---

### 阶段五：验收检查

**云伍检查清单**:
- [ ] 前台所有页面完整
- [ ] 后台管理界面完整
- [ ] 响应式适配正常
- [ ] 动效正常
- [ ] 与API联调通过

**云陆检查清单**:
- [ ] 所有API正常
- [ ] 后台登录正常
- [ ] 后台功能可操作
- [ ] 数据库设计完整
- [ ] 与前端联调通过

---

## 📋 完整项目组成

一个完整的网站必须包含：

| 模块 | 内容 | 负责 |
|------|------|------|
| **前台** | PC端 + 手机自适应 | 云伍 |
| **后台** | 管理系统 (CMS) | 云伍 + 云陆 |
| **数据库** | 完整的数据表设计 | 云陆 |
| **API** | 前后端接口 | 云陆 |

---

## 🎯 质量控制要点

### 1. 需求阶段 (云捌)
- 必须包含前后台完整设计
- 必须有数据库设计
- 必须有API设计

### 2. 设计阶段 (云肆)
- 必须有UI设计标准
- 必须有完整UI设计图
- 必须有动效说明

### 3. 开发前 (云伍/云陆)
- 必须Review需求和设计
- 不符合必须反馈，不能盲目开发

### 4. 开发阶段 (云伍/云陆)
- 前台：PC + 移动端完整
- 后台：CMS完整可操作
- 后端：API + 数据库完整

### 5. 验收阶段
- 自检查完整性
- 联调测试
- 功能可操作性验证

---

## 🚨 关键规则

1. **不符合不开发**: 云伍/云陆发现需求和设计不符，必须停止并反馈
2. **完整才交付**: 前台必须PC+移动端完整，后台必须CMS完整可操作
3. **自检查**: 开发完成后必须自我检查完整性
4. **可操作验证**: 后台功能必须验证可正常操作

## 📁 项目文件结构标准

```
项目名/
├── docs/                          # 文档目录
│   ├── PRD.md                     # 需求文档 (云捌)
│   ├── UI-Design.md               # UI设计文档 (云肆)
│   └── API.md                     # API文档 (云陆)
│
├── database/                      # 数据库
│   ├── init.sql                   # 初始化脚本
│   └── admin.sql                  # 管理员数据
│
├── frontend/                      # 前端 (云伍)
│   ├── src/
│   │   ├── views/
│   │   │   ├── Home.vue          # 前台页面
│   │   │   └── ...
│   │   └── admin/                # 后台管理
│   │       ├── Login.vue
│   │       └── Dashboard.vue
│   └── ...
│
└── backend/                       # 后端 (云陆)
    ├── src/
    │   ├── controller/           # API控制器
    │   ├── service/              # 业务逻辑
    │   ├── entity/               # 数据库实体
    │   └── repository/           # 数据访问
    └── ...
```
