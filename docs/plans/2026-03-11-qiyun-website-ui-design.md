### 设计细节

**Hero区域**:
- 背景: 135度渐变 (0f172a → 1e3a8a → 1d4ed8)
- 高度: 100vh (全屏)
- 主标题: 48px, 白色, 字重Bold
- 副标题: 20px, 白色/80透明度
- CTA按钮: 主按钮(蓝底白字) + 次按钮(透明边框白字)
- 装饰: 微妙的粒子动画或几何图形

**服务卡片**:
- 布局: 3列等宽, gap 24px
- 卡片: 白底, 圆角12px, 阴影card
- 图标区: 48×48px, 蓝底, 圆角8px, 白色图标
- 标题: H4, 深色
- 描述: 14px, 次要文字色
- 链接: 蓝色, 箭头图标
- Hover: 上浮4px, 阴影加深

**案例卡片**:
- 布局: 横向滚动或3列网格
- 图片: 16:9比例, 圆角8px顶部
- 标签: 绝对定位在图片上, 毛玻璃效果
- 标题: 16px, 字重600
- 客户名: 14px, 次要色

## 通用组件规范

### 导航栏
- 高度: 64px
- 背景: 毛玻璃效果 (rgba(15,23,42,0.8) + blur)
- Logo: 32px图标 + 文字
- 链接: 14px, 白色/80%透明度
- 激活状态: 底部2px蓝色下划线
- 移动端: 汉堡菜单, 全屏抽屉

### 页脚
- 背景: 深色 (#0f172a)
- 布局: 4列 (Logo+简介, 快速链接, 服务, 联系)
- 文字: 白色/60%透明度
- 链接: Hover变白色
- 底部: 版权信息 + 备案号

### 页面标题区
- 背景: 渐变或纯色
- 高度: 200-300px
- 标题: H1, 白色或深色
- 面包屑: 小字, 路径导航

---

## 响应式适配要点

### 桌面端 (≥1024px)
- 最大宽度: 1200px, 居中
- 导航: 水平展开
- 网格: 3-4列
- 字体: 标准尺寸

### 平板端 (768-1023px)
- 边距: 24px
- 导航: 水平但紧凑
- 网格: 2列
- 字体: 略微缩小

### 移动端 (<768px)
- 边距: 16px
- 导航: 汉堡菜单
- 网格: 1列
- 字体: 适当缩小
- 触摸目标: 最小44px
